# Prairies Planner

Public, sanitized static PWA for Camping Les Prairies de la Mer and nearby Port Grimaud / Grimaud / Saint-Tropez planning.

## Privacy Boundary

This app intentionally excludes:

- Booking references
- Payment, deposit, tax, or private arrival details
- Personal notes from `notes-while-there.md`
- Private contact or itinerary specifics
- Non-public travel dates

## Local Preview

```bash
python3 -m http.server 8080
```

Then open `http://127.0.0.1:8080`.

## GitHub Pages

Push this folder as the root of a public GitHub repo, then enable GitHub Pages from the `main` branch root.

On iPhone: open the GitHub Pages URL in Safari, tap Share, then Add to Home Screen.
